# RiskAuth 2FA

Веб-приложение на Python с регистрацией, входом и двухфакторной проверкой через TOTP.

## Запуск

```bash
python app.py
```

Открыть:

```text
http://127.0.0.1:5000
```

## Что есть

- регистрация пользователя;
- PBKDF2-хэш пароля;
- TOTP-секрет при регистрации;
- ввод 6-значного кода при входе;
- кабинет пользователя.
