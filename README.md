# RiskAuth 2FA

Веб-приложение на Python с регистрацией, входом и хранением пароля в виде хэша.

## Запуск

```bash
python app.py
```

Открыть:

```text
http://127.0.0.1:5000
```

## Что добавлено

- регистрация пользователя;
- вход по логину и паролю;
- PBKDF2-хэш пароля с солью;
- простая сессия через cookie;
- кабинет пользователя.
